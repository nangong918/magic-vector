**Magic Vector**
================


# 背景

【魔法向量】MagicVector: 
在2019年的时候爱上了Anki Vector，一直想要复刻但是没有机会。
2022年AI元年，出现了OpenAI，让一切成为可能。
2024年上半年推出MCP，实现语言AI大模型能进行本地函数调用。
2024年下半年Spring推出SpringAI，让Java开发能快速进行AI应用层开发。用于能让梦想圆满。

# 介绍

基本功能: 
1. App创建Agent, Agent记忆存储在后端.
2. Esp32-SC作为嵌入式设备, App选择创建的Agent注入给嵌入式设备.
用户可以通过App直接聊天, 通过嵌入式设备聊天, 也可以通过App控制Esp32设备
